# 128. 最长连续序列

## 题目描述

   给定一个未排序的整数数组，找出最长连续序列的长度。

    要求算法的时间复杂度为 O(n)。

## 示例:
```
   输入: [100, 4, 200, 1, 3, 2]
    输出: 4
    解释: 最长连续序列是 [1, 2, 3, 4]。它的长度为 4。
```

## 思路介绍

### 方法一：排序

#### 题目解析


#### 思路

1. 对数组进行排序；
2. 定义结果 res = 1， 计数 temp_count =1
3. 遍历数组：
   1. 如果 nums[i] - nums[i-1] > 1 
      1. temp_count = 1
   2. 如果 nums[i] - nums[i-1] == 1
      1. temp_count += 1
   3. 判断当前 temp_count > res:
      1. res = temp_count
4. 返回结果

   
#### 复杂度计算

> 时间复杂度：O(logn)
>  
> 空间复杂度：O(n)
