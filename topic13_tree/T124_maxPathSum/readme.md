# 104. 二叉树的最大深度

## 题目
### 介绍 

    给定一个二叉树，找出其最大深度。

    二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

    说明: 叶子节点是指没有子节点的节点。

### 示例：
    给定二叉树 [3,9,20,null,null,15,7]，

     3
    / \
    9  20
      /  \
     15   7
    返回它的最大深度 3 。

## 解法

### 方法一 递归法

#### 思路

1. 递归的求解左右子树各自的最大深度

2. 最后返回两者中最大的深度加上根节点的 1 即为最大深度。


##### 复杂度计算

> 时间复杂度：O(n)

> 空间复杂度：O(logn)
