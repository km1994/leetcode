# 剑指 Offer 53 - II. 0～n-1中缺失的数字

## 题目描述

    一个长度为n-1的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围0～n-1之内。在范围0～n-1内的n个数字中有且只有一个数字不在该数组中，请找出这个数字。

    示例 1:

    输入: [0,1,3]
    输出: 2
    示例 2:

    输入: [0,1,2,3,4,5,6,7,9]
    输出: 8

## 思路介绍

### 方法一: 二分法

#### 思路

1. 定义 left 和 right;
2. 遍历数组 ：left<=right：
   1.  mid = (left+right)//2
   2. if mid == nums[mid]：left = mid+1
   3. else :right = mid - 1
3. 返回 left 

#### 复杂度计算

时间复杂度：O(logN)
空间复杂度：O(1)
